Picture-in-picture (PiP) is a mode where the video is shown in a window independently of the browser. Browsers have different PiP capabilities and the options available differ.

## Picture-in-Picture Types

- Browser native - the browser's proprietary PiP implementation that is not controllable by API
- Video element PiP 
- Document PiP
